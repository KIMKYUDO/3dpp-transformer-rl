1+3+4=8 => 1/3/4로 분리할 수는 없음(스칼라) 대신
x = u(√2, √2, 0) + v(-√2, √2, 0) + w(0, 0, 1)은 서로 다른 서브 스페이스를 갖기에 W_Q(같은 부분 공간이 되도록 학습하는)를 곱하면 분리 가능, W_Q가 u와 같은 부분일 경우 xW_Q = (u+v+w)*W_Q = (u+0+0)*W_Q
"u와v와w가 독립 스페이스이다"가 핵심 
직교가 아니더라도 빗사영으로 분리가능

3차원 비직교 예시 (직접 분리됨)

U = span{a}, a = (1, 0, 0)

V = span{b}, b = (1, 1, 0)  ←  a·b = 1 (직교 아님)

W = span{c}, c = (0, 0, 1)

S = span{b, c}. 여기서 R^3 = U ⊕ S,  U ∩ S = {0}.

이제 P를 U로의 빗사영으로 잡으면 (한 구현 예):

P = a α^T,   α = (1, -1, 0).

검증:
P a = a,
P b = 0,
P c = 0

⇒  ∀x = u + βb + γc,   P x = u.

즉, 직교가 아니어도 v, w 성분이 정확히 0으로 사라지고 u만 남습니다.
